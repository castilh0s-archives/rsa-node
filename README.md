# RSA Node.js

## Como rodar

- Instalar as dependências

```bash
yarn install
```

- Descriptografar

```bash
node decrypt.js <CHAVES_PRIVADAS> <TEXTO_ENCRIPTADO> <DESTINO>
```

- Encriptografar

```bash
node encrypt.js <CHAVES_PUBLICAS> <TEXTO> <DESTINO>
```
